## Create Project

```
npx create-react-app app-name

npm start
```

<br>

Install tailwind

```
npm install -D tailwindcss
npx tailwindcss init

```

[Install Tailwind CSS with Create React App](https://tailwindcss.com/docs/guides/create-react-app)

<br>

Install React Icons

```
npm add react-icons
```
